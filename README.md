## Problem Statement
Many citizens, especially in rural and semi-urban areas, are unaware of government welfare schemes they are eligible for. Information is scattered, complex, and difficult to understand.

## Solution
JanSaathi is a web-based platform that helps users easily discover government welfare schemes based on their basic details through a simple and user-friendly interface.

## Key Features
- Simple UI for easy access
- Scheme discovery support
- Modern frontend architecture
- Secure backend integration

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- Supabase
